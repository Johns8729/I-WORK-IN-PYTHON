# 📦 Simple Sales System in Python

This project is a small **Python program** that simulates recording sales in a store. It allows the user to enter products, their price, and the quantity sold, calculates the total for each sale, and finally shows a summary of the day.

## 📁 Project Structure

The project is divided into three files:

* **main.py** → Main file that runs the program and records sales.
* **calcular.py** → Contains the function that calculates the total price of a sale.
* **mensaje.py** → Displays the final summary of all sales made during the day.

## ⚙️ How It Works

1. The program asks for:

   * Product name
   * Product price
   * Quantity sold

2. It calculates the total of the sale.

3. The sold products are stored in a list.

4. The program allows the user to continue registering sales until the user types **"no"**.

5. At the end, a **daily summary** is displayed with all the products sold and the total amount collected.
