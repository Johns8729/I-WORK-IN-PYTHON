# Project Title
Daily Store Sales Record System

Simple overview of use/purpose.

This program allows a user to register product sales in a a store, keep track of quantities sold, and calculate the total money earned during the day.

---

# Description

This project is a simple Python program that simulates a daily sales registration system for a store.

The program asks the user to enter the product name, unit price, and quantity sold. It then calculates the total value of the sale and adds it to the daily revenue.

All products sold are stored in a dictionary so the program can track how many units of each product were sold.

At the end of the process, the program generates a daily summary showing:
- The products sold
- The quantity sold for each product
- The total amount of money earned during the day

This project demonstrates the use of Python fundamentals such as dictionaries, loops, conditional statements, and user input.

# Getting Started

## Dependencies

To run this program you need:

- Python 3 installed
- A code editor or terminal

Example operating systems:

- Windows 10 or newer
- macOS
- Linux

---

## Installing

1. Download the Python file containing the program.
2. Save it in a folder on your computer.
3. Open a terminal or command prompt in that folder.

No external libraries are required because the program only uses built-in Python features.

---

## Executing program

To run the program follow these steps:

1. Open the terminal or command prompt.
2. Navigate to the folder where the file is saved.
3. Run the program using Python.

Example command:

```bash
python main.py
```
